A custom WooCommerce plugin to hide prices for specific products and show a custom message.

== Description ==
This plugin allows WooCommerce store owners to hide prices for selected products and display a custom message, like "Price available upon request."

== Features ==
- Hide prices for specific products.
- Display a customizable message instead of the price.
- Disable "Add to Cart" button for these products.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Edit the product IDs in the plugin settings file as needed.
